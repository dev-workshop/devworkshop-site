MEAN Stack Development

JavaScript has become one of the most common programming languages in the world, and with the advent of Node.js, it's becoming even more ubiquitous. Meet the MEAN Stack: MongoDB, Express, AngularJS, and Node.js. In this session, you will learn fundamentals of all four technologies and how they relate to each other in web architecture. From database to user interface, learn how to write great web apps completely in JavaScript.

What's Covered

  1. Setting up your environment
  2. Node.js server & npm package manager
  3. MongoDB and understanding NoSQL databases
  4. Building a web API with Express
  5. Interacting with MongoDB from Express with Mongoose
  6. Building an interactive UI with AngularJS
  7. Summary, questions, and wrap-up
